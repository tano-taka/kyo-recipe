# アプリ名

## 「KYOレシピ」（投稿・検索サイト）


# 概要

- **レシピの検索/詳細確認**  
利用者は投稿されたレシピを自由に見ることができ、自分が作りたいレシピを検索することができます。一覧画面から画像をクリックをすることで詳細の確認が可能です。

- **ユーザー詳細確認**  
レシピの投稿者名をクリックすると、投稿者の「投稿数」「総合いいね数」「１レシピでの平均材料費」とその投稿者のレシピ一覧が表記されます。

- **ユーザー登録機能**  
ユーザーを登録することで、レシピ投稿と評価機能(いいねボタン)を利用することが可能となります。

- **レシピの投稿**  
ログイン者は画像付きのレシピを投稿することができます。自分が作ったレシピを画像付きで投稿し、一覧に表示することができます。

- **評価機能（いいねボタン）**  
ログインをすることで、レシピに対して「いいね」の評価が可能となります。「いいね」はレシピ毎に押された回数が表記される為、良いレシピの判断が可能です。

# 制作背景（意図）
**「料理レシピサイトの料理はおしゃれすぎる！」というのが出発点です。**  

まずペルソナとしまして、男性、大学生〜３０代の一人暮らし。長い時間働いたり（勉強）しているもののお金はなく自炊。代わり映えのしないメニューを毎日食べている。
そんな生活の中でも、簡単で、安くて、美味しいものを食べたい（作りたい）と考えている人。  

上記の方の多くは、家に豊富な調味料はなく、備蓄している食材も限られていることが多いと考えます。そんな中、有名な料理レシピサイトで検索すると、美味しそうな料理が多く掲載されているものの、その分手の込んだ料理や、当たり前に豊富な調味料や食材を隠し味に使ったりというレシピが散見されます。一人暮らしであれば、カップ麺や缶詰、インスタント食材など、安く手に入る食材で作りたいはず。見た目はいまいちでも、味と手軽さと価格を重視したレシピサイトの方が良いはず。そのように考えて作成したのがこのアプリです。

# 工夫したポイント

- トップ画面でレシピ毎に「いいね」された回数を表記することで良いレシピを見つけやすくしました。  

- 投稿されたレシピは新着順で表記される仕組みとし、投稿意欲をあげることと、内容が変化しやすい状況を作りました。  

- トップ画面では必要な情報だけを載せるため、レシピの手順１までの表記とし、画像をクリックすることでレシピ詳細へ遷移するようにしました。  

- レシピ詳細画面では「いいね」ボタンを非同期通信で実装しているため違和感なくクリックが可能です。  

- ユーザー詳細画面へ遷移し、ユーザーがもらった「いいね」の総合計を出すことで投稿意欲を高めると共に優良な作者を見つけやすくしました。  

- また、同ページにてレシピの平均材料費を出すことで、重要な要素である価格を意識してもらう仕組みとしました。  

- 投稿画面では文字数制限をして簡単なレシピを意識してもらうことと、簡単で良いというメッセージを伝え、投稿のハードルを下げています。

# 課題や今後実装したい機能

- TOP画面での人気レシピ表記  
レシピ毎の「いいね」回数が多い順でTOP画像への表記をすることで人気のレシピをわかりやすくする為に実装します。表記は新着のすぐ下辺りに人気順として表記、TOP6レシピを表記予定です。  

- 食材検索  
持っている材料をチェックボックスで選択することにより、その食材で作成できるレシピを表記してくれる検索機能を実装します。アクティブハッシュを利用して選択肢をもうけて、投稿時に使える食材に関しても制限をかける予定です。  

- 調味料検索  
持っている調味料をチェックボックスで選択することにより、その調味料で作成できるレシピを表記してくれる検索機能を実装します。アクティブハッシュを利用して選択肢をもうけて、投稿時に使える調味料に関しても制限をかける予定です。  

- インスタント食材検索  
持っているインスタント食材をチェックボックスで選択することにより、そのインスタント食材で作成できるレシピを表記してくれる検索機能を実装します。アクティブハッシュを利用して選択肢をもうけて、投稿時に使える調味料に関しても制限をかける予定です。  

- レシピ投稿時のプレビュー表記  
レシピ投稿時の画像添付にてプレビュー出るように実装します。間違った画像の添付防止となります。JQueryを使っての非同期処理で実装予定です。  

- 文字登録時の残文字数の表記  
レシピ投稿時の手順を打ち込み時に残りの文字数が順次表記されるように実装します。文字数制限がある為、確認しながら打ち込めることでユーザビリティが向上します。JQueryを使っての非同期処理で実装予定です。  

- レシピに対してのコメント機能  
レシピ詳細画面に利用者からのコメント覧を実装します。評価精度の向上と投稿意欲の向上や、コミュニティーを作る上で重要な要素となります。JQueryを使っての非同期通信で実装予定です。
